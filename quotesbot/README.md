Quotes Bot
===============
A simple bot that uses scrapy to crawl the [Website](http://quotes.toscrape.com/) to extract:

- quotes
-  author
-  tags


Install methods:

    pip install scrapy

Run:

    scrapy crawl quotes -o quotes.json

Resources:
  [Scrapy Docs](https://doc.scrapy.org/en/latest/intro/install.html)
